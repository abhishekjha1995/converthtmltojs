Need to install this package globally and run the following command

To convert Single HTML file to js
chjfile [source html file path] [destination js file path] [variable name in js] [(optional) true/false wether html body tag is required or not]

To convert all HTML/htm file in a folder to js
chjfolder [source folder path] [destination folder path] [variable name in js] [(optional) true/false wether html body tag is required or not]
